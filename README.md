## Project Overview

HRFlow is a Human Resource Management System designed to simplify and automate essential HR operations including employee management, attendance tracking, leave processing, and workforce analytics.

The system helps organizations reduce manual HR workload and improve operational efficiency through centralized data management and real-time insights.

---

## Key Features

- Employee onboarding and management
- Attendance check-in/check-out system
- Leave request and approval workflow
- HR analytics dashboard
- Department-wise workforce insights
